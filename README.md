# Wanderlust

A responsive travel website built from provided mockups, focused on translating design into a working layout.

---

## Overview

Wanderlust is a front-end project where the goal was to recreate a travel website design using CSS. The project emphasizes responsive design and layout accuracy across multiple screen sizes.

---

## Features

- Fully responsive layout (mobile, tablet, desktop)  
- Mobile-first CSS approach  
- Clean layout based on provided mockups  
- Consistent spacing, typography, and structure  

---

## Tech Stack

- HTML  
- CSS  
- GitHub Pages (deployment)  

---

## How It Works

- The HTML structure was provided  
- CSS was written from scratch to match the design mockups  
- Media queries were used to adjust layouts across different screen sizes  
- The layout was tested across devices to ensure consistency  

---

## What I Learned

- How to translate a static design into a working website  
- Writing responsive CSS using a mobile-first approach  
- Managing layout, spacing, and typography across breakpoints  
- Testing and refining designs across different screen sizes
